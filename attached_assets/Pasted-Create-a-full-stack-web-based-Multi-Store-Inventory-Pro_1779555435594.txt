Create a full-stack web-based **Multi-Store Inventory, Production & Sales Management System** for a manufacturing company. The system must track materials from raw material purchase up to final finished product sales while monitoring wastage, production efficiency, inventory movement, approvals, and financial transactions.

# 1. User Roles & Permissions

Implement role-based access control with different permissions for:

* Super Admin
* Store Manager
* Production Manager
* Sales Officer
* Finance Officer
* Approver / General Manager

Each role should only access permitted modules and actions.

---

# 2. Store Structure & Workflow

The company operates multiple connected stores:

1. Main Raw Material Store
2. Semi-Finished Production Mini Store
3. Final Production Mini Store
4. Main Finished Product Store

The system must track item movement step-by-step between all stores.

## Main Workflow

* Raw materials are purchased and received into the Main Raw Material Store.
* Materials are transferred to the Semi-Finished Production Store.
* Semi-finished products are processed and transferred to the Final Production Store.
* Final products are transferred to the Main Finished Product Store.
* Final products are sold to customers.

The system must track:

* Quantities moved
* Wastage/loss at each stage
* Production outputs
* Remaining balances
* Batch tracking
* Inventory history
* Item movement logs

---

# 3. Direct Packaging Material Workflow

The system must also support direct item requests from the Main Finished Product Store to the Main Raw Material Store specifically for packaging materials.

Packaging materials may include:

* Cartons
* Bottles
* Caps
* Labels
* Stickers
* Plastic bags
* Wrappers
* Sealing materials
* Packaging accessories

## Packaging Workflow

1. Finished Product Store creates packaging material request.
2. Main Raw Material Store reviews request.
3. Request approved or rejected.
4. Shipment initiated.
5. Finished Product Store confirms receipt.

The system must:

* Distinguish packaging materials from production raw materials.
* Track packaging material usage separately.
* Deduct packaging inventory automatically.
* Link packaging materials to finished products where applicable.
* Include packaging material usage in wastage and inventory reports.

---

# 4. GRN (Goods Receiving Note) Module

The Main Raw Material Store Manager should:

* Create GRN whenever goods are received from suppliers/vendors.

GRN form should include:

* Supplier company name
* Invoice number
* Purchase order number
* Delivery note number
* Date received
* Item name
* Quantity
* Unit
* Unit cost
* Total cost
* Batch number
* Expiry date (if applicable)
* Warehouse/store location
* Attachments/documents upload

## Approval Workflow

* GRN must go through manager approval before payment is processed.
* Finance department can only process payments after approval.

GRN statuses:

* Draft
* Pending Approval
* Approved
* Rejected
* Paid

Generate printable GRN documents and PDF exports.
